# ☢️ S.T.A.L.K.E.R. 2: Heart of Chornobyl — MultiHack

[English](README.md) | **Русский**

## Описание

Многофункциональный Internal Cheat для **S.T.A.L.K.E.R. 2: Heart of Chornobyl** с ImGui Overlay, поддержкой DLSS/NVIDIA Streamline, локализацией RU/EN, системой профилей, горячих клавиш, ESP, Aimbot, спавнером предметов, телепортом и другими возможностями.

> **Версия чита:** v3.4

> **Поддерживаемая версия игры:** v1.9 (Build 22554680) — Ultimate Edition

> **Только для одиночной игры.**

## Возможности

### 🎯 AIM
- Aimbot (FOV, Smooth, выбор кости, клавиша)
- **Мультиточка** — приоритет костей (голова → шея → грудь → таз)
- **Проверка видимости** — не целится сквозь стены
- **Упреждение** — автоматический вынос по движущейся цели. Скорость пули берётся из самой игры, для каждого ствола своя — настраивать нечего
- **Автосмена цели** — мгновенно переводит прицел, когда текущий враг убит
- **Триггербот** — авто-выстрел, когда видимая цель попадает под прицел (FOV, задержка реакции, опциональная клавиша). Работает и без аимбота
- Silent Aim
- **Без покачивания в прицеле** — убирает качание оружия при прицеливании
- Настройка FOV

### 💪 Exploits
- God Mode
- Бесконечные здоровье, броня, выносливость, патроны
- **Без перелома ног** — падения с высоты больше не калечат и не замедляют (при бесконечном здоровье)
- **Без отдачи** — камеру не подбрасывает, и пули летят точно в марку (при зажиме их больше не уводит вверх)
- Нет разброса, перезарядки, заклинивания
- One Shot
- Множитель урона
- Скорострельность
- **Нулевой вес** — нет перегруза; залипшее замедление от перевеса снимается сразу при включении
- **Иммунитет к эффектам монстров** — защита от контроля: сбивание с ног (псевдогигант), оглушение (кровосос), тряска камеры, отбор оружия (бюрер), блок действий/ввода
- Нет голода, радиации, кровотечения
- Fly / NoClip / Infinite Jump
- Заморозка времени
- **Настройка фонарика** — полный контроль цвета, яркости, дальности, ширины и объёмного луча + пресеты (Прожектор / УФ / Ночное виденье)
- Множитель подбираемого лута
- Невидимость (стелс-режим)
- Нет опьянения
- Нет желания спать
- Нет сонливости
- Нет пси-воздействия
- Купоны
- **Редактор репутации фракций** — показывает репутацию к каждой фракции и позволяет её задать (кнопки по фракциям + глобальные «Все дружелюбны / нейтральны / враждебны»)
- Задать репутацию NPC в радиусе
- Разблокировка всех заметок
